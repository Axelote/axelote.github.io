Продвижение сайта

Чтобы люди видели и заходили на сайт в интернете, он должен находиться в выдаче поисковых систем (Google, других). У сайта есть 2 способа туда попасть

Контекстная реклама

Рекламные обьявления показывается перед поисковой выдачей (сразу в топе!)

- мгновенный результат
- плата за каждого посетителя

SEO продвижение

Сайт попадает в топ 10-20 поисковой выдачи

- результат 4-6 месяцев
- бесплатный приток посетителей на сайт
